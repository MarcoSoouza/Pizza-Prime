🍕 Pizzaria Prime

Sistema desenvolvido para gerenciamento de uma pizzaria, com foco em pedidos, cardápio e controle de clientes. Este projeto foi criado como atividade acadêmica para aplicar conceitos de desenvolvimento de software.

📌 Descrição

A Pizzaria Prime é uma aplicação que permite:

Visualizar o cardápio de pizzas
Realizar pedidos online
Gerenciar clientes
Acompanhar status dos pedidos
Administrar produtos e vendas (área administrativa)

🎯 Objetivo

O objetivo do projeto é simular um sistema real de pizzaria, aplicando conceitos como:

CRUD (Create, Read, Update, Delete)
Integração entre front-end e back-end
Manipulação de banco de dados
Organização de código e boas práticas

🛠️ Tecnologias Utilizadas

Exemplo (ajuste conforme seu projeto):

Front-end: HTML, CSS, JavaScript
Back-end: Java / Python
Banco de Dados: MySQL
Ferramentas: Git, GitHub, Vscode

💻 Como Executar o Projeto
Clone o repositório:
git clone https://github.com/MarcoSoouza/Pizza-Prime/tree/main
Acesse a pasta:
cd pizzaria
Instale as dependências:
npm install
Execute o projeto:
npm start
Acesse no navegador:
[http://localhost:3000](https://marcosoouza.github.io/Pizza-Prime/index.html)
